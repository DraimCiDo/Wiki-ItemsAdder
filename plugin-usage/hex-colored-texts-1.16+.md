# Текст с использованием HEX-цветов \(1.16+\)

## Как использовать на 1.16 HEX-цвета

Вы можете изменить цвет текста и задать пользовательский HEX-цвет, используя специальную нотацию: `%#000000%`.

Например:

**display\_name: "`%#FE5A00%`magic\_fuel"**

```yaml
  magic_fuel:
    display_name: "%#FE5A00%magic_fuel"
```

{% hint style="info" %}
Он работает везде, в том числе и в файлах перевода плагина.
{% endhint %}
