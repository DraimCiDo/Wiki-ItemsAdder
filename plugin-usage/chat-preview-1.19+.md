### Включение предварительного просмотра чата

Откройте `config.yml` ItemsAdder и включите эту функцию, затем запустите `/iareload` и присоединитесь к серверу.

{% code title="config.yml" %}
```yaml
chat-preview: true
```
{% endcode %}
