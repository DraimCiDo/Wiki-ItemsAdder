# Починка на наковальне

## Пример

```yaml
anvil_repair:
    emerald_sword:
        permission: itemsadder.emerald_sword
        ingredient: EMERALD
        item: itemsadder:emerald_sword
```

В данном примере я создал рецепт `anvil_repair` под названием `emerald_sword`.
